\doxysubsection*{Esecuzione}

Prima di eseguire il file è necessario unzippare la cartella.~\newline
 L\textquotesingle{}eseguibile si trova nella cartella \char`\"{}\+Debug\char`\"{} ed è chiamato \char`\"{}\+Caso\+\_\+di\+\_\+\+Studio-\/\+Carpool.\+exe\char`\"{}.~\newline
 Per poter funzionare, l\textquotesingle{}eseguibile ha bisogno della cartella chiamata \char`\"{}\+Files\char`\"{} che ha il compito di contenere i file inerenti ai conducenti, ai viaggi, alle valutazioni, agli ID e alcuni file temporanei che si autocancelleranno dopo il loro utilizzo (se il programma verrà chiuso mentre il loro utilizzo non è terminato, i file non saranno cancellati, e di conseguenza potrebbero nascere errori).~\newline
 Un file temporaneo sarà creato durante la prenotazione del viaggio, questo file ha il compito di contenere gli identificatori univoci dei viaggi validi per la prenotazione.~\newline
 Il secondo file temporaneo sarà creato durante l\textquotesingle{}ordinamento dei conducenti o dei viaggi.~\newline
 Se la cartella \char`\"{}\+Files\char`\"{} non esiste il programma creerà una nuova cartella \char`\"{}\+Files\char`\"{} nella root del progetto, il programma si comporterà nello stesso modo anche per quanto riguarda i file contenuti nella cartella. \doxysubsection*{Documentazione}

Tutti i documenti del progetto si trovano nella cartella \char`\"{}\+Documentation\char`\"{}.
